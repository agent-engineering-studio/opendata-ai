# Code of Conduct

opendata-ai is an open-source civic technology project. We want contributors, users and maintainers to collaborate in a respectful, practical and evidence-based way.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but be clear when something is wrong.
- Keep discussion focused on the project, data quality, reproducibility and user value.
- Welcome newcomers and help them find small, useful contributions.
- Credit sources, datasets and external work.

## Unacceptable behavior

- Harassment, insults, personal attacks or discriminatory language.
- Publishing private information, secrets or personal data.
- Repeated off-topic promotion or spam.
- Misrepresenting data sources, licenses or project capabilities.
- Pressuring maintainers for unpaid urgent work.

## Enforcement

Maintainers may edit, hide or remove comments, close issues, block users, or restrict participation when behavior harms the project or community.

## Scope

This code applies to project spaces including GitHub issues, pull requests, discussions, documentation and community channels connected to opendata-ai.
